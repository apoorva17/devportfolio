Projects

RepoSense (Final Year Project)

RepoSense is a contribution analysis tool for Git repositories. It is currently being used for software engineering modules in NUS. I implemented features to improve the usefulness and usability of the tool, and reviewed Pull Requests submitted by developers who wanted to contribute code to the tool.
Some of the key features I implemented include: Breakdown by file type, Breakdown by groups (docs, functional-code, test-code, other), Breakdown by blank line contribution and Glob Filter.

Technologies used: Java, Jade, Vue.js, Javascript

Smart Living Hackathon

When my friends and I participated in the Smart Living Hackathon, we were tasked with exploring how voice technology can transform the way people live in under 24 hours. With 35 teams working on the same problem, we had to ensure our solution was creative enough to capture the attention of the judges. After many intensive brainstorming sessions and pitches to the various partners of the hackathon, we developed a working prototype which we demonstrated to a panel of judges and live audience and won “Best Hack”!

Find My Soulmate!

Find My Soulmate! is a matchmaking service which identifies the user’s match based on how compatible they are with someone. The service determines the user’s personality based on his/her past behaviour on Facebook and uses it to find a match with other users. The application was hosted on AWS and uses IBM's Personality Insights Services to determine the user's personality.

Technologies used: AWS, IBM Watson, Facebook Graph API, Vue.js, Javascript, HTML

Exploitation of MIFARE Classic Card

As part of an academic security project, we explored the possibility of sniffing and modifying data stored in a MIFARE Classic 1K Exploit Card using a NFC reader. The motivation behind this project was to find out how secure NFC transactions are given the prevalence of NFC devices used in contactless transactions.
